# InvestAI

**InvestAI** — это продвинутый веб-терминал для прогнозирования стоимости акций с использованием технологий машинного обучения (ML) и искусственного интеллекта. Проект объединяет стильный пользовательский интерфейс, похожий на профессиональные торговые терминалы, с локальной моделью предсказания на Python (FastAPI + Scikit-Learn) и возможностью облачного анализа через Gemini.

## 🚀 Что уже реализовано

1. **Современный Web-Интерфейс (React + Vite)**
   - Интерактивный дашборд с графиками цен (на базе `recharts`).
   - Имитация реального торгового терминала с плавными анимациями, индикаторами трендов и аналитикой.
   - Возможность переключения между различными активами (AAPL, GOOGL, TSLA, NVDA, MSFT) и временными периодами.
   - Журнал телеметрии прогнозов (история успешных и неуспешных предсказаний).

2. **Локальное ML-ядро (Python + FastAPI)**
   - Бэкенд-сервер на **FastAPI**, который обрабатывает запросы от веб-интерфейса.
   - Модель `GradientBoostingRegressor` из `scikit-learn`, которая динамически обучается на данных графика, выделяя лаговые признаки (цены за предыдущие дни) для предсказания цены на следующий день.
   - Сервер рассчитывает не только будущую цену, но и определяет тренд (Bullish/Bearish), вычисляет уверенность в процентах (Confidence) и генерирует текстовое резюме.

3. **Гибридный выбор движка прогнозирования**
   - Возможность переключаться между **"Своей (Local)"** моделью, работающей на вашем ПК, и **"Облаком Gemini"** для текстового анализа.

## 💻 Как запустить проект

Для полноценной работы вам потребуется запустить **оба** сервера: фронтенд и бэкенд.

### 1. Запуск ML-Бэкенда (Python)

Бэкенд отвечает за локальные математические вычисления и предсказания.

1. Убедитесь, что у вас установлен Python (версии 3.8+).
2. Активируйте виртуальное окружение:
   ```bash
   venv\Scripts\Activate
   ```
3. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите сервер FastAPI:
   ```bash
   uvicorn backend.main:app --reload
   ```
*Сервер запустится по адресу `http://localhost:8000`.*

### 2. Запуск Веб-интерфейса (React)

Фронтенд отвечает за визуализацию и взаимодействие.

1. Откройте **новое окно терминала** (оставив работать бэкенд в первом окне).
2. Установите зависимости Node.js (если запускаете впервые):
   ```bash
   npm install
   ```
3. (Опционально) Если вы хотите использовать облако Gemini, добавьте ваш `GEMINI_API_KEY` в файл `.env.local`.
4. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
*Терминал запустится по адресу `http://localhost:3000` (или другому порту, указанному в консоли).*

## 💡 Использование

1. Откройте приложение в браузере.
2. В панели "Конфигурация параметров" выберите интересующую вас акцию (например, Microsoft).
3. В разделе "Ядро модели" выберите **Своя (Local)**.
4. Нажмите кнопку **"Запустить анализ"**.
5. График мгновенно обновится: интерфейс отправит текущие данные на ваш Python-бэкенд, который проведет анализ, обучит модель на лету и вернет предсказание на следующий день вместе с расчетом тренда!
